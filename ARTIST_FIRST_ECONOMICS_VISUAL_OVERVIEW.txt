Why These Amounts?

SHARING (0.5 tokens):
  - Easy action (1-click share)
  - Requires user agreement to share
  - Creates positive network effects
  - Bonus if recipient engages (+1-2 tokens)

LISTENING (1.0 + bonuses):
  - Time investment required (listen to full/partial song)
  - Completion bonuses incentivize quality
  - 50% listen = 1.25 tokens
  - 75% listen = 1.50 tokens  
  - 90% listen = 2.00 tokens
  â†’ Encourages artists to create engaging content

VOTING (0.5 tokens):
  - Advisory only (not binding, prevents vote-buying)
  - Encourages civic participation
  - Matching majority = +0.1 bonus
  - But only base if voting against majority
  â†’ Participatory governance, not financial incentive

BANDWIDTH (2.0 + bonuses):
  - Requires infrastructure investment
  - Ongoing cost (electricity, data)
  - Per 100MB served = +0.5 tokens
  - Per unique listener = +0.1 tokens
  â†’ Incentivizes network participation

REFERRAL (1.0 + bonuses):
  - Drives platform growth
  - Referenced user first purchase = +0.5 bonus
  - Ongoing activity = +0.2/month bonus
  â†’ Aligns with platform success

   Artists created the content. They deserve maximum value.
   100% on primary sales. 80% on secondary sales.

2. USER REWARDS TIED TO EFFORT
   Rewards proportional to engagement.
   Sharing < Listening < Bandwidth (effort required)

3. QUALITY INCENTIVES
   Completion bonuses reward engaging content.
   Better songs = higher listen completion = higher artist earnings

4. SUSTAINABLE ECONOMICS
   No Ponzi schemes. No unlimited token inflation.
   Rewards are real (come from user engagement, not thin air)

5. COMMUNITY PARTICIPATION
   Voting is advisory (not binding).
   Users have voice, but artists maintain control.
